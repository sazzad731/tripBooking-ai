# TripBooking AI Project by Sazzad Hossen

This is a frontend travel booking landing page built with Next.js 16, React 19, TypeScript, and Tailwind CSS.

live link: https://tripbooking-ai.vercel.app/


## Setup Instructions

1. Make sure you are using `Node.js >= 20.9.0`.
2. Install dependencies:

```bash
git clone https://github.com/sazzad731/tripBooking-ai.git
```

```bash
npm install
```

3. Start the development server:

```bash
npm run dev
```

4. Open `http://localhost:3000` in your browser.

Optional checks:

```bash
npm run lint
npm run build
```

## Assumptions

- The assignment is implemented as a static marketing homepage without backend/API integration.
- Images used in the UI are served from the local `public/` folder.
- No environment variables are required for local setup.

## Features Implemented

- Responsive homepage built with reusable Next.js components
- Header and hero section
- Trusted partners section
- Popular destinations section
- Featured tours carousel
- Special trip offers section
- Top rated hotels carousel section
- AI travel deals promo section
- Sponsor/payment partners strip
